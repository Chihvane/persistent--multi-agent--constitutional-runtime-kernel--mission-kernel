## Repository Policy

This public repository does not commit the raw reference PDF corpus by default. The archive contains full arXiv PDFs and web snapshots that are useful as a private evidence pack, but public redistribution can introduce licensing and copyright ambiguity.

Instead, this repository records:

- the package checksum
- the source manifest
- the citation-to-claim mapping
- the public URLs and arXiv IDs needed for verification

Raw PDFs can be restored locally from the source archive when doing citation review.
