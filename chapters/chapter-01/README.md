## Core Terms

- **Agent OS**: a single-agent operating model where one assistant holds the conversational state, chooses tools, performs work, and decides when a task is complete.
- **Constitutional Runtime**: a governed execution environment where agents operate under explicit rules, capabilities, accountability structures, and closure criteria.
- **Task Ownership**: the runtime-level assignment of responsibility for a unit of work.
- **Legitimate Closure**: a verifiable end state where the system can show what was done, by whom, under which authority, with what evidence, and what risk remains.
