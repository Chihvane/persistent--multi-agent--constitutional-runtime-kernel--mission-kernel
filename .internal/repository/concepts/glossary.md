# Glossary

## Agent OS

A single-agent operating model in which one primary agent manages intent, planning, tools, execution, memory, and closure through a conversational loop.

## Constitutional Runtime

A governed execution environment where agents operate under explicit, inspectable, and enforceable rules.

## Mission Kernel

The runtime layer that preserves durable user intent and mission state separately from transient agent execution.

## Task Ownership

The assignment of responsibility for a work unit to an actor, including scope, authority, expected evidence, and closure obligations.

## Scoped Capability

A permission grant limited by actor, task, artifact, action, environment, and time.

## Audit Trail

Structured event history that records decisions, actions, authority context, affected artifacts, and evidence.

## Legitimate Closure

A verifiable task end state supported by evidence, checks, residual risk disclosure, and acceptance criteria.
