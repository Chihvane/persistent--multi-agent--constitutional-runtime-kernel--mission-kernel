# Citation Evidence

Chapter 1 is backed by a public citation evidence package.

## Included Public Files

- `chapter1/references/README.md`
- `chapter1/references/citation-evidence-index.md`
- `chapter1/references/reference-package-manifest.md`

These files map claims to sources and document the public evidence basis for the
chapter.

## Raw Corpus Policy

The raw reference PDF corpus is intentionally not committed by default.

Reasons:

- avoid ambiguous redistribution of third-party PDFs
- keep the public repository lightweight
- make the public evidence layer inspectable without bundling every source file

If a future release includes source snapshots, each file must have clear
redistribution permission and must be listed in the reference manifest.

## Evidence Update Rule

Any material change to a cited claim should update the evidence index in the
same pull request.
