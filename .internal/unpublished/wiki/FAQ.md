# FAQ

## Is this repository the implementation?

No. This is the public mission, article, and whitepaper repository. The private
implementation/source repository remains separate.

## Is the project open source?

Not by default. The repository is publicly readable, but reuse rights are
controlled by `.internal/repository/legal/LICENSE.md` and any license notices in specific release packages.

## Which Chapter 1 file should I cite?

Cite the matching canonical PDF:

- Chinese: `chapter1/final-draft-v3/package-v3/chapter1.pdf`
- English: `chapter1/final-draft-v3/package-v3-en/chapter1.pdf`

## Why are raw reference PDFs not included?

The public repository includes citation indexes and evidence manifests, but not
the raw reference PDF corpus by default. This avoids unclear redistribution of
third-party materials.

## Can I contribute corrections?

Yes. Open a documentation issue or pull request. Corrections to cited claims
should include source context and update the evidence index when needed.
