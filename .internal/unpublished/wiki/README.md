# Wiki Source

This directory contains the source pages for the GitHub Wiki.

GitHub stores wiki pages in a separate `*.wiki.git` repository, so these files
are kept here as the version-controlled canonical draft. When the public
repository is ready, copy these Markdown files into the GitHub Wiki repository.

- [Home](./Home.md)
- [Sidebar](./_Sidebar.md)
- [Public Release Checklist](./Public-Release-Checklist.md)
