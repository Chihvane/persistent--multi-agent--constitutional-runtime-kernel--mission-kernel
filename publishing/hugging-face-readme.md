# From Single-Agent OS to Constitutional Runtime

This project documents the design of a governed multi-agent runtime: a constitutional execution environment for long-horizon AI work.

The source implementation remains private for now. This public repository contains the mission narrative, article series, and whitepaper drafts.

## Core Claim

Long-horizon AI work does not fail only because models make mistakes. It fails because the surrounding runtime often has no durable task ownership, no permission semantics, no audit trail, and no legitimate definition of closure.

The project explores how to rebuild AI work around governed runtime primitives instead of a single unbounded assistant loop.
